# SportOn - Dashboard Admin Toko 

Repository ini berisi source code untuk **SportOn**, dashboard admin untuk toko perlengkapan olahraga. Website ini dibuat untuk mempermudah pengelolaan stok barang dan transaksi harian.

Project ini adalah submission saya, **Naufal Rizky Putera**, untuk program **FutureMakers YEP PLAN**.

---

## ï¿½ Tentang Project Ini

Intinya, aplikasi ini dibuat supaya pemilik toko bisa ngurusin data-data penting tanpa pusing. Di sini admin bisa:
- Input dan update stok produk baru.
- Bikin kategori barang biar rapi.
- Cek transaksi yang masuk dari customer.
- Atur info rekening bank toko.

## ğŸ› ï¸ Tech Stack

Teknologi yang saya pakai di sini dipilih biar development-nya efisien dan hasilnya maksimal:

- **Next.js 15 (App Router)** & **TypeScript**: Kombinasi ini bikin performa kenceng dan kode lebih aman dari bug.
- **Tailwind CSS**: Urusan styling jadi lebih cepet dan pastinya responsif di semua ukuran layar.
- **Zustand**: Buat state management yang ringan dan nggak ribet (lebih simpel dari Redux).
- **React Hook Form**: Validasi input data jadi jauh lebih gampang dan rapi.
